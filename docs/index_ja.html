北陸新幹線

富山-東京間を業界最速で空席照会します。

説明を隠す
「北陸新幹線」は、富山-東京間の列車の時刻表および空席状況を照会するアプリです（お住まいの地域に応じて、金沢-上野間などへ変更できます）。

迅速な情報照会
アプリを起動した時点で本日の列車の空席状況を照会します。初期表示は、現在地が富山駅に近ければ「上り」、東京駅に近ければ「下り」が表示されます。

全ての列車を表示
ご指定日の全ての列車を一覧表示します。当日分については、走行中の列車情報も表示します。

走行地点の確認
端末の位置情報から現在の走行区間をポイントします。時刻表よりも列車が遅れていば、遅延していることを表示します。

JR 東日本のサービス「えきねっと」では、新幹線の路線、出発・到着駅、および日時の指定を繰り返して照会する必要があるため、予約までに時間を要してしまいがちです。
本アプリを使用することにより、少ない操作で簡単に情報を照会できることができます。本アプリで事前に空席情報を確認することでスムーズな列車予約ができるでしょう。

本アプリを使用するには、Pythonista 3 を別途インストールしたうえで GitHub から直接ソースファイルをダウンロードしてご利用ください（現在公開停止中）。
